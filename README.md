# Installation Guide

## NVM Installation

``` bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
```

## Node.js 18 Installation

``` bash
nvm install 18
nvm use 18

# Install dependencies
npm install
```

## Start Development Server

``` bash
npm run dev
```

## Build for Production

``` bash
npm run build
```
